### Установка

1. **Клонирование репозитория**:

   ```bash
   git clone https://github.com/timuritodev/test-rmh.git
   cd test-rmh

2. **Установка библиотек**:

   ```bash
    npm install

3. **Запуск веб-приложения**:

   ```bash
    npm run dev
